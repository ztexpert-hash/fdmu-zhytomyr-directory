Довідник зареєстрованих оцінок ФДМУ — Житомир
Версія сайту: 1.9

Додано:
- імпорт Excel-файлу ФДМУ прямо в браузері;
- формування нового calc_data.json без Python;
- підрахунок імпортованих записів, квартир, кімнат/гуртожитків і виконавців;
- виправлено відображення версії у футері.

Як оновити базу:
1. Відкрити сайт.
2. Натиснути «Імпорт Excel ФДМУ».
3. Обрати Excel-файл ФДМУ.
4. Натиснути «Завантажити calc_data.json».
5. Замінити calc_data.json у GitHub-репозиторії.
6. Commit changes.
