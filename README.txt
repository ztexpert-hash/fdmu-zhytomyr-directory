Довідник зареєстрованих оцінок ФДМУ — Житомир

Статичний сайт для GitHub Pages.

Файли для репозиторію:
- index.html
- calc_data.json
- manifest.json
- sw.js

Як опублікувати:
1. Створити новий GitHub repository.
2. Завантажити ці файли в корінь репозиторію.
3. Settings → Pages → Deploy from branch → main / root.
4. Відкрити посилання GitHub Pages.

Призначення:
Довідкова статистика вже зареєстрованих оцінок ФДМУ по квартирах у місті Житомирі.
Не є звітом про оцінку і не формує офіційні аналоги.
