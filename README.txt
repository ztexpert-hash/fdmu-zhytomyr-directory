Інструкція v6.0 Auto Update
---------------------------
1. Розпакувати архів у робочу папку.
2. За потреби встановити залежності: pip install -r requirements.txt
3. Запустити Оновити_ФДМУ_v6.bat або python fdmu_auto_update_v6.py
4. Скрипт сам знайде ZIP на сайті ФДМУ, витягне Obekty_Zhytlovoi_neruxomosti, відфільтрує місто Житомир і сформує output_update/fdmu_zhytomyr_directory_update.zip.
5. На GitHub завантажувати саме ZIP із папки output_update.

Важливо:
- Статуси Зареєстровано і Перевірено включаються в базу.
- Місто Житомир = Житомир + Богунський + Корольовський.
- Житомирський район не включається.
- Якщо у ФДМУ з’являться нові корисні поля, вони будуть показані в output_update/update_report.txt.
