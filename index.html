Призначення
Інструмент показує, які вартості вже реєструвалися у ФДМУ по квартирах у Житомирі та новобудовах Оліївки 2018+. Це довідник для попереднього інформування клієнта, а не автоматична оцінка і не є автоматично придатним джерелом для звіту без окремого аналізу оцінювачем.
Додаткові модулі
Основний екран залишено чистим. Деталі, список оцінок і аналітика відкриваються тільки за потреби.
Імпорт Excel ФДМУ
Виберіть Excel-файл ФДМУ. Сайт сформує новий calc_data.json по Житомиру: Богунський, Корольовський, записи без району; Оліївка враховується у складі міської зони. Після завантаження JSON потрібно замінити файл у GitHub і натиснути Commit changes.
