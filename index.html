Призначення
Інструмент показує, які вартості вже реєструвалися у ФДМУ по квартирах у місті Житомирі. Це довідник для попереднього інформування клієнта, а не автоматична оцінка і не джерело офіційних аналогів для звіту.
Імпорт Excel ФДМУ
Виберіть Excel-файл ФДМУ. Сайт сформує новий calc_data.json тільки по місту Житомир: Богунський, Корольовський та записи без району. Після завантаження JSON потрібно замінити файл у GitHub і натиснути Commit changes.
